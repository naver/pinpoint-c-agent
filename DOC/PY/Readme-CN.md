### 安装步骤

#### 搭建 pinpointPy 模块

1. 安装python虚拟环境，请参考
https://packaging.python.org/guides/installing-using-pip-and-virtual-environments/

2. 回到根目录下，安装 pinpointPy（建议在Python虚拟环境下执行）
```shell
$ python setup.py install
```
